# Wesley Gavitt Portfolio Homepage

## Overview

This project is a personal portfolio homepage created for a JavaScript course assignment. The page includes an About section, Featured Content section, dynamically rendered Projects section, Contact form, custom welcome modal, tooltip, dark mode toggle, and browser storage features.

## Languages and Technologies Used

- HTML
- CSS
- JavaScript
- DOM manipulation
- localStorage
- sessionStorage
- JSON.stringify()
- JSON.parse()

## Project Features

- Custom welcome modal that appears when the page loads
- Close button for dismissing the modal
- Dark mode toggle that saves the user's preference with localStorage
- Skills list generated dynamically with JavaScript
- Three custom JavaScript project objects
- Project object array stored in sessionStorage
- Project data converted to a string with JSON.stringify()
- Project data parsed back into JavaScript objects with JSON.parse()
- Project cards rendered dynamically on the page
- Tooltip on the contact form submit button
- Contact form prevents page refresh and displays a delayed confirmation message

## Files Included

- index.html
- styles.css
- script.js
- README.md

## How to Run the Project

1. Download or clone the project folder.
2. Make sure all files are in the same folder.
3. Open index.html in a web browser.
4. Use the navigation links, dark mode toggle, modal close button, project cards, and contact form to test the page.

## Dependencies

This project does not require any external libraries or frameworks. It runs with standard HTML, CSS, and JavaScript in a modern web browser.
